# **Customer Churn Analysis**
<br>

This project is analyses customer churn data to identify pattern why customers are leaving the services and build predictive models which type
of customers will leave the company.
<br>

This is my first project in Data analysis and building Predictive models.
<br>

While doing this project i applied what i learnt and tried get as many insights and outputs from it.The process include
- Data loading, Basic data manipulation, Details regarding columns
- Data visualization : Making insights out of graphs, bar chats etc..
- Data cleaning : Dealing with null, duplicate values, encoding categorical values ( label encoding)
- Model building: Basic models like Logistic regression, Decision Tree, Random Forest
- Evaluation : Comparing model performance using accuracy, recall score , precision score and F1 score.
<br>

## Reflections
- I realized the importance of **data preprocessing** — it often takes more effort than model building and very important to get a model with good predictive capacity.
- Sharing the notebook on GitHub taught me version control and collaboration skills.
<br>

To run the notebook locally, install the following:
- Python 3.8+
- Jupyter Notebook / JupyterLab
- Libraries:
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- imbalanced-learn (for SMOTE)
- Dataset - link **https://www.kaggle.com/datasets/rashadrmammadov/customer-churn-dataset**
<br>

How to Run
- Install dependencies: `pip install -r requirements.txt`
- Open `notebook.ipynb` in Jupyter.
